55. Pop-up dialogs

Tkinter provides three modules that can create pop-up dialog windows for you:

  • Section 55.1, €œThe tkMessageBox dialogs module€, provides an assortment of common pop-ups for simple tasks.

  • Section 55.2, €œThe tkFileDialog module€, allows the user to browse for files.

  • Section 55.3, €œThe tkColorChooser module€, allows the user to select a color.
